Portfolio

This map visualizes the spatial distribution of CVS and Walgreens pharmacy locations against Detroit’s population density. Using gridded population data and geocoded store locations, it highlights the greater coverage of CVS (14) compared to Walgreens (7). This map was created in ArcGISpro.

A symbolic and labeled global map showcasing countries with historically severe earthquakes. This learning map was created to practice cartographic design, including symbology, labeling, and visual hierarchy in QGIS.

This project involved georeferencing a scanned plat map and manually digitizing parcels over current basemaps. The goal was to reconstruct housing parcel boundaries with precision using QGIS geoprocessing and digitization tools.

This comparative map visualizes the change in Rouge River’s flow by overlaying historical aerial imagery with digitized paths and performing a contour analysis on the modern DEM. It demonstrates the spatial impact of river channelization using QGIS.

This map displays the exact locations of vending machines on campus. Field data was collected using the Mergin Maps mobile app and synced through the Mergin Cloud platform, and the points were visualized in QGIS.

Using a hexbin spatial analysis technique, this map quantifies the density of vending machines on campus to reveal clustering patterns and popular areas. Built entirely in QGIS with graduated color density styling.

This map uses DEM elevation data to perform a contour analysis of terrain around the University of Michigan-Dearborn, identifying low-lying zones likely to support vernal pools. An environmental application of QGIS-based terrain modeling.

A hydrological risk assessment map identifying potential flood zones and vernal pool formations using the Bluespot plugin in QGIS. The analysis highlights water-retaining depressions, aiding ecological planning and hazard prevention.
